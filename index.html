Что мы хотим от e2e?

  • Высокая скорость выполнения тестов
  • Проволить тесты на различных платформах
  • Качественные репорты и логи о проделанной работе

Почему же в облаке?

  • Быстрота выполнения
    Тест-план на различных конфигурациях возможно запускать параллельно. Но есть лимит
  • Большой выбор окружений
    Более сотен комбинаций браузеров/устройств/ОС
  • Легко конфигурить
    Настраивается буквально парой строчек кода
  • Не надо настраивать ВМ

Как можно приготовить?

  • Настраиваем protractor.config.js
  • Скачиваем Sauce Connect
  • Запускаем Sauce Connect
  • Запускаем тесты
  • Результат

exports.config = {

 specs: ['test/**/*.spec.js'],
 sauceUser: 'YOUR_NAME',
 sauceKey: 'OUR_NAME:YOUR_KEY',

 multiCapabilities: [
    {
        'browserName': 'chrome',
        'version': '51.0',
        'platform': 'Windows 10',
        'name': 'Chrome v.51.0'
    }
 ]
}

Улучшаем рецепт

  • Добавляем в настройки конфигурации параметр shardTestFiles: true
  • Добавляем библиотеку для работы с api
    npm i saucelabs --save-dev
  • Добавляем тесты в "Билд".
    //Protractor.conf.js
params: {
	souceLab: new SauceLabs({
		username: 'YourName',
		password: 'YourKey'
	}),
	build: 'The buildest'
}
//Any spec file
afterAll((done) => {
	browser.params.souceLab.updateJob(sessionId, {
		build: browser.params.build
	}, done);
});
